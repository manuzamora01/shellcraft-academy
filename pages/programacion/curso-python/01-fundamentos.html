# Briefing del Módulo

Python es el lenguaje por excelencia para la automatización de tareas, el análisis de datos y la creación de exploits en ciberseguridad. En esta primera fase, dominarás la sintaxis básica y aprenderás a comunicarte con la máquina.

1. Tu primer comando: Print

Para que nuestro script nos devuelva información, utilizamos la función nativa print(). En Python, las cadenas de texto (strings) siempre deben ir entre comillas simples o dobles.

# Script: hola_mundo.py
print("Inicializando sistema...")

2. Variables y Tipos de Datos

Una variable es un espacio en memoria donde guardamos un dato. Python es de tipado dinámico, lo que significa que no tienes que decirle qué tipo de dato es (texto, número, etc.); él lo deduce solo.

# Declaración de variables
objetivo = "192.168.1.50" # Tipo String (Texto)
puerto = 8080 # Tipo Integer (Número entero)

# Concatenación (unir texto y variables)
print("Escaneando objetivo: " + objetivo)
💡 Pro-Tip: Puedes usar f-strings para inyectar variables en el texto más fácilmente. Ejemplo: print(f"Puerto {puerto} abierto").
