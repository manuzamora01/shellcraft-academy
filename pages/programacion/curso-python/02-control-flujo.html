02. Control de Flujo

Condicionales (If/Else) y Bucles (For/While) en Python

# Briefing del Módulo

Un script lineal es predecible y limitado. Para crear herramientas potentes (como un escáner de red, un ataque de diccionario o un script de respuesta a incidentes), necesitas que tu código tome decisiones basadas en los datos que recibe y que repita tareas dinámicamente.

Objetivos principales:
  • [*] Tomar decisiones lógicas con if, elif y else.
  • [*] Iterar sobre colecciones estáticas con bucles for.
  • [*] Ejecutar tareas condicionadas y continuas con bucles while.

1. Condicionales (if / elif / else)

Usamos if para evaluar si una condición es verdadera (True). Si lo es, se ejecuta el bloque indentado. Si es falsa (False), podemos usar else para ejecutar otra acción, o elif para probar una condición alternativa.

puerto = 22

if puerto == 22:
    print("[+] Posible servicio SSH detectado.")
elif puerto == 80:
    print("[+] Posible servicio HTTP detectado.")
else:
    print("[-] Puerto desconocido.")

2. Bucles FOR (Automatización definida)

El bucle for es ideal cuando sabes exactamente cuántas veces quieres repetir algo. Por ejemplo, recorrer un rango del 1 al 3 usando range(1, 4).

for puerto in range(1, 4):
    print(f"Escaneando puerto {puerto}...")

3. Bucles WHILE (Ejecución condicional)

El bucle while se ejecuta continuamente mientras una condición siga siendo verdadera. Es muy útil para realizar reintentos o mantener un script "escuchando" indefinidamente. ¡Debes asegurarte de modificar la variable de control dentro del bucle para no crear un bucle infinito que cuelgue el sistema!

intentos = 0

while intentos < 3:
    print(f"Reintentando conexión... Intento {intentos + 1}")
    intentos += 1 # Aumentamos el contador en 1 en cada vuelta
💡 Pro-Tip: En Python la indentación (los espacios a la izquierda) no es estética, es obligatoria. Un bloque mal indentado arrojará un error fatal de sintaxis.

🎯 Misión 02: Defensa contra Fuerza Bruta

Escribe un script en el Sandbox que simule un sistema de login defensivo combinando un bucle while y un condicional if/else.

Reglas:
1. Inicia una variable intentos = 1.
2. Crea un bucle while que se ejecute mientras los intentos sean menores o iguales a 3.
3. Dentro del bucle, usa un if: si los intentos llegan exactamente a 3, imprime "[!] ALERTA CRÍTICA: Bloqueando IP por demasiados intentos."
4. Si no (usando else), imprime "[-] Contraseña incorrecta. Intento X" (donde X es el número del intento).
5. ¡No olvides sumar 1 a la variable intentos en cada vuelta para salir del bucle!

Esta es la estructura clásica de un sistema de bloqueo por intentos:

intentos = 1

while intentos <= 3:
    if intentos == 3:
        print("[!] ALERTA CRÍTICA: Bloqueando IP por demasiados intentos.")
    else:
        print(f"[-] Contraseña incorrecta. Intento {intentos}")
    
    intentos += 1
