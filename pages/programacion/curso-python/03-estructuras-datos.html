03. Estructuras de Datos

Listas y Diccionarios

# Briefing del Módulo

Trabajar con variables sueltas no es eficiente cuando manejas grandes volúmenes de información. Las estructuras de datos te permiten agrupar valores relacionados para procesarlos en masa.

Objetivos principales:
  • [*] Almacenar múltiples elementos en una Lista.
  • [*] Acceder y modificar datos mediante índices.
  • [*] Organizar información mediante pares clave-valor con Diccionarios.

1. Listas (Colecciones Ordenadas)

Una lista es un conjunto ordenado de elementos. Se definen usando corchetes []. El primer elemento siempre está en la posición 0.

servicios = ["ssh", "http", "ftp"]

# Acceder al primer elemento
print(servicios[0]) # Imprime: ssh

# Añadir un nuevo elemento
servicios.append("dns")

2. Diccionarios (Clave-Valor)

Los diccionarios funcionan como una base de datos pequeña. Cada dato se guarda asociado a una clave única. Se definen con llaves {}.

servidor = {
    "ip": "10.0.0.1",
    "so": "Ubuntu 22.04",
    "status": "activo"
}

# Acceder a un valor por su clave
print(servidor["ip"]) # Imprime: 10.0.0.1
