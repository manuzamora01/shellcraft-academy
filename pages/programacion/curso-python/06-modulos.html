# Briefing del Módulo

Python viene con "pilas incluidas". En lugar de programar desde cero funciones matemáticas complejas, interacciones con el sistema operativo o algoritmos de cifrado, puedes usar import para cargar herramientas ya fabricadas por otros ingenieros.

El poder del Import

Para ciberseguridad, librerías como os (sistema operativo) y hashlib (criptografía) son el pan de cada día.

import os
import hashlib

# Obteniendo información del sistema operativo usando 'os'
usuario_actual = os.getlogin() # Solo funciona en entornos locales, no en el sandbox

# Cifrando una contraseña usando 'hashlib'
clave = "admin123"
hash_md5 = hashlib.md5(clave.encode()).hexdigest()
print(f"Hash MD5: {hash_md5}")

🎯 Misión 06: El Criptógrafo

Importa la librería hashlib. Transforma la palabra "ShellCraft" en un hash SHA-256 (usando hashlib.sha256()) e imprímelo por consola. No olvides hacer el .encode() y el .hexdigest() como en el ejemplo.

import hashlib

palabra = "ShellCraft"
hash_final = hashlib.sha256(palabra.encode()).hexdigest()
print(f"El hash SHA-256 es: {hash_final}")
