Operación: Generador de Claves

📄 Briefing de la Misión

Para la fase de post-explotación, necesitamos crear usuarios temporales con contraseñas seguras y aleatorias para mantener persistencia sin levantar sospechas.

Instrucciones:
  • Importa el módulo random.
  • Crea una lista llamada palabras con 3 palabras que tú elijas.
  • Crea un bucle for que se repita 3 veces (usa range()).
  • En cada vuelta del bucle, selecciona una palabra aleatoria de tu lista usando random.choice(palabras), añádele un número aleatorio del 1 al 99 usando random.randint(1, 99), e imprímela por pantalla.
import random

palabras = ["Admin", "Servidor", "Root"]

for i in range(3):
    palabra_elegida = random.choice(palabras)
    numero_aleatorio = random.randint(1, 99)
    password = f"{palabra_elegida}{numero_aleatorio}"
    print(f"Contraseña generada: {password}")
