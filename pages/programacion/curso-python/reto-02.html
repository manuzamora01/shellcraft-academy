Nivel 2: SysAdmin

Operación: Filtrado de Logs

📄 Briefing de la Misión

Hemos exportado un archivo de registro del servidor firewall. Hay cientos de conexiones, pero solo nos importan las que fallaron. Escribe un script para automatizar la auditoría.

Instrucciones:
  • Primero, usa el Sandbox para crear un archivo llamado firewall.log y escribe en él varias líneas (ej: "Conexion OK de 192.168.1.5", "ERROR: Intento fallido de 10.0.0.9", "Conexion OK de 172.16.0.1").
  • Vuelve a abrir el archivo en modo lectura ("r").
  • Recorre el archivo usando un bucle for linea in archivo:.
  • Usa un if para comprobar si la palabra "ERROR" está dentro de la línea. Si está, imprímela.
# 1. Creamos el archivo de prueba simulando el entorno
with open("firewall.log", "w") as f:
    f.write("Conexion OK - IP: 192.168.1.1\n")
    f.write("ERROR: Credenciales invalidas - IP: 10.0.0.5\n")
    f.write("Conexion OK - IP: 192.168.1.2\n")
    f.write("ERROR: Intento de inyeccion SQL - IP: 10.0.0.9\n")

# 2. Leemos y filtramos
print("[!] Iniciando auditoría de logs...\n")
with open("firewall.log", "r") as log:
    for linea in log:
        if "ERROR" in linea:
            print(f"ALERTA ENCONTRADA: {linea.strip()}") # .strip() quita saltos de linea extra
