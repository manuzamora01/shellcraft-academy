Operación: Cracker de Hashes

📄 Briefing de la Misión

Hemos interceptado un hash SHA-256 de una base de datos enemiga. Creemos que la contraseña es débil y se encuentra dentro de un diccionario conocido. Tu misión es crackear el hash y descubrir la contraseña en texto claro.

Hash Objetivo: 5e884898da28047151d0e56f8dc6292773603d0d6aabbdd62a11ef721d1542d8
Instrucciones:
  • Importa el módulo hashlib.
  • Crea una lista llamada diccionario con las siguientes palabras: ["123456", "admin", "password", "qwerty"].
  • Recorre el diccionario con un bucle for.
  • Convierte cada palabra del diccionario a SHA-256 usando hashlib.sha256(palabra.encode()).hexdigest().
  • Compara el resultado con el Hash Objetivo usando un if. Si coincide, imprime la contraseña encontrada y usa la palabra break para detener el bucle y no seguir buscando.
import hashlib

hash_objetivo = "5e884898da28047151d0e56f8dc6292773603d0d6aabbdd62a11ef721d1542d8"
diccionario = ["123456", "admin", "password", "qwerty"]

print("[*] Iniciando ataque de diccionario...")

for palabra in diccionario:
    # Generamos el hash de la palabra actual
    intento_hash = hashlib.sha256(palabra.encode()).hexdigest()
    
    if intento_hash == hash_objetivo:
        print(f"[+] HASH CRACKEADO: La contraseña es '{palabra}'")
        break # Detiene el bucle porque ya hemos ganado
    else:
        print(f"[-] Fallo: '{palabra}' no es la contraseña.")
